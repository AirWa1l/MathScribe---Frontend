# Capturas de la interfaz desplegada

Evidencia visual del entregable de diseño. Se toman sobre el despliegue público,
no sobre el entorno local, para que documenten el sistema tal como lo ve un
evaluador.

| Archivo | Qué debe mostrar |
|---|---|
| `01-inicio.png` | Pantalla inicial con las dos vías de captura |
| `02-camara.png` | Cámara activa con la vista previa |
| `03-resolucion.png` | LaTeX reconocido, resultado y pasos |
| `04-metricas.png` | Panel de métricas desplegado con cifras reales |
| `05-error.png` | Un estado de error, por ejemplo permiso denegado |

Recomendaciones: ventana a un ancho de escritorio normal, sin extensiones del
navegador visibles, y con datos reales en el panel de métricas (basta con
reconocer y resolver una expresión antes de capturar).
